A semester-long project for INFO 4320 Rapid Prototyping and Physical Computing.

Musical Cubes are fun and interactive devices that allow users to create music by moving cubes in different positions over a colour gradient. Users can select the different instruments by using a touch sensor. The cubes use a colour sensor to detect the colour it is placed on, then translates this RGB value into a MIDI event. The cube consists of a colour sensor, capacitive touch sensor, LED ring, and is controlled by an Adafruit 32u4 board which uses a 500mAh battery and Bluetiith Low Energy to communicate remotely with a computer. This project combines the knowledge of hardware, software, design and mechanics.

Project for INFO 4120 Ubiquitous Computing.

PAUSE-ture is a head and neck posture correction system that aims to improve the posture of users who spend extended periods of time working at their desks using design principles based on calm technology and mindless computing. The system is designed as a wearable device that attaches onto over-ear headphones. The system provides subtle audio and visual feedback to notify the user to correct his or her posture when poor posture is detected. The goal of the PAUSE-ture project is to provide an unobtrusive posture correction and training system which targets forward head posture. This started as a semester-long project for INFO 4120 Ubiquitous Computing, but we are continuing to conduct further research into this area of study to discover the possibilities of audio feedback.

Group project for INFO3300 Data-Driven Web Applications.

Using Javascript D3 and Google Search Data, we created a static data visualization of search interest in selected fashion brands across the USA. We collected the data from Google and analyzed different geographical trends over the country.

Final Project for INFO3300 Data-Driven Web Applications.

We created a data visualization of gender trends in various academic disciplines from the 60's onwards. The project uses enrollment rates for different college majors organized by gender. The female to male ratio is represented by a colour gradient in each blob.
